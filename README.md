# Electric_Vehicle_analysis
data analytics project showcasing Power BI dashboard design and Ev market insights

## 📊 Overview
This project explores electric vehicle population data to uncover trends in adoption, manufacturer market share, vehicle types, and eligibility for clean alternative fuel programs.

## 🗂️ Data Source
Washington State Electric Vehicle Population dataset (Kaggle)

## 🛠️ Tools Used
- Power BI (data modeling, DAX, dashboard design)
- Data cleaning and transformation prior to visualization

## ✨ Key Features
- **Interactive filters:** City, Electric Utility, Electric Vehicle Type
- **KPIs:** Total vehicles, average electric range
- **Breakdowns:** BEV vs PHEV split, top manufacturers, top models, model year trends, geographic distribution, CAFV (Clean Alternative Fuel Vehicle) eligibility

## 🔍 Key Insights
- Tesla leads the market with 35 vehicles, followed by Chevrolet and Ford (16 each)
- BEVs account for 63% of vehicles vs 37% PHEV
- Vehicle adoption sharply increased from 2018 onward, peaking at 22 units in the most recent model year
- Nearly half (47.69%) of vehicles qualify as Clean Alternative Fuel Vehicles
